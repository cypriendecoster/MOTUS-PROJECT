# MOTUS

Jeu de devinette de mots developpe en HTML, CSS et JavaScript vanilla.

## Apercu
- Jeu jouable directement dans le navigateur
- Clavier virtuel et clavier physique
- Historique des parties
- Deploiement automatique configure pour GitHub Pages

## Technologies
- HTML5
- CSS3
- JavaScript

## Lancer le projet
Ouvre `index.html` dans ton navigateur.

## Publication GitHub Pages
Le projet contient deja le workflow `.github/workflows/deploy-pages.yml`.
Une fois pousse sur GitHub, chaque `git push` sur `main` redeploiera le site.
